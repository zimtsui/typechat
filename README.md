# TypeChat

[![NPM Version](https://img.shields.io/npm/v/@zimtsui/typechat?style=flat-square)](https://www.npmjs.com/package/@zimtsui/typechat)

TypeChat 是一个强类型的 LLM 推理服务商 API 适配器。

## 支持服务商 API 类型

-   OpenAI ChatCompletions
-   OpenAI Responses
-   Google
-   Anthropic
